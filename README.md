## 🚀 About Me

> *"Building scalable systems, one commit at a time."*

I'm **Hayford Afriyie**, a **Senior Software Engineer** from **Ghana 🇬🇭** with expertise spanning **35+ programming languages** and modern full-stack architectures. I design, build, and deploy production-grade applications across web, mobile, cloud, and systems programming.

- 🔭 Currently architecting **[EDSPiKE](https://edspike.com)** — an EdTech platform transforming education in Africa
- 🌱 Exploring **Rust systems programming**, **WebAssembly**, and **AI/ML integration**
- 💬 Ask me about **distributed systems**, **microservices**, **DevOps**, or **performance optimization**
- ⚡ Fun fact: I once debugged a production outage at 3 AM using only a terminal and coffee ☕
